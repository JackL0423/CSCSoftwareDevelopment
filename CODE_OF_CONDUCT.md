# Code of Conduct

## Our Commitment

This project is part of our CSC305 capstone course at the University of Rhode Island. We are committed to providing a welcoming and inclusive environment for everyoneâ€”whether you're a classmate, instructor, or community member reviewing our work.

## Standards

**Encouraged behaviors**:
- Being respectful and professional
- Providing constructive feedback
- Accepting different viewpoints and approaches
- Taking responsibility for mistakes and learning from them
- Supporting fellow contributors

**Unacceptable behaviors**:
- Harassment or discriminatory language
- Personal attacks or insults
- Publishing private information without consent
- Trolling or deliberately disruptive behavior
- Any conduct inappropriate for an academic setting

## Reporting Issues

If you experience or witness unacceptable behavior:

1. Address it directly with the individual if you feel comfortable
2. If the issue persists, contact the project maintainers
3. All reports will be handled confidentially and fairly

We recognize that this is a learning environment. Most issues can be resolved through respectful dialogue. Repeated or serious violations may result in removal from the project.

## This Applies To

- GitHub issues and pull requests
- Project discussions
- Anywhere you're representing this project

Basically: if you're interacting with this project, please be respectful!

## Purpose

This code of conduct supports our goal of creating a positive learning environment where everyone can contribute effectively and professionally.
