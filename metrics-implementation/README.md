# D7 Repeat Recipe Rate - Production Code

Production code for D7 retention metric implementation.

## Custom Actions (FlutterFlow)
- `checkAndLogRecipeCompletion.dart` - Main completion tracking
- `initializeUserSession.dart` - Session initialization
- `checkScrollCompletion.dart` - Scroll detection helper

## Cloud Function (Firebase)
- `calculateD7Retention.js` - Daily retention calculation

Deploy these to FlutterFlow and Firebase respectively.
